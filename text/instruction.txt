ברוכים הבאים לחדר הבריחה - מחוץ לכדור מקווים שתהנו!מומלץ להצטייד בדף ובכלי כתיבה כדי לרשום רמזים ולחשב פתרונות.<span class = "headline">ניווט</span>בחדר תנווטו באמצעות העכבר. בצדדי המסך יהיו כפתורים שיאפשרו לכם לפנות לחלקים אחרים בחדר.הכפתורים יופיעו רק כאשר העכבר עליהם ויראו כך:אם תעבירו את העכבר על חלק מהפריטים בחדר, תופיע לכם זכוכית מגדלת.תכלו ללחוץ עליה כדי להסתכל עליהם יותר מקרוב. זכוכית המגדלת תיראה כך:אחרי שתלחצו עליה יופיע בפינה הימנית התחתונה כפתור שיאפשר לכם לחזור בחזרה.הכפתור חזרה (שמופיע גם בדף זה) נראה כך:מומלץ להסתובב בכל החדר לפני שמתחילים לפתור.<span class = "headline">סיסמאות</span>בחלקים מסויימים בחדר תידרשו להזין סיסמה.כדי להזין את הסיסמה, יש להקליד את הסיסמה באמצעות המקלדת ולאחר מכן ללחוץ על Enter במקלדת.במידה והסיסמה שהזנתם נכונה יפתח לכם הרמז הבא. במידה והסיסמה שגויה, הסיסמה תעלם, ותופיע לכם הודעהבתחתית המסך שהסיסמה שגויה. שימו לב - כל סיסמה שגויה תוריד ניקוד!<span class = "headline">רמזים</span>השמאלית העליונה של הדף מופיע כפתור רמז שנראה כך:במידה ונתקעתם תוכלו להשתמש בו. כמובן ששימוש ברמזים מוריד מהניקוד הסופי. ניתן לראות את עלות הרמז לפני שפותחים אותו.ניתן לצפות שוב ברמז שכבר נפתח ללא עלות נוספת.<span class = "headline">טוב לדעת</span>בחדר יש סיסמאות וניסויים אנלוגים שקשורים למאדים. כדי שגם אתם תוכלו לדעת עליהם דאגנו שתופיע הודעה שתפרט עליהם בתחתית המסך.חשוב לציין שהעובדות האלו הן לא רמזים ולא משפיעות על פתרון החדר בכלל. הן נועדו רק כדי להוסיף להנאה ולידע שלכם.<span class = "headline">ניקוד</span>הניקוד נקבע לפי כמה דברים:1) הזמן שעבר - ככל שיקח לכם יותר זמן לפתור את החדר תקבלו ניקוד נמוך יותר.2) תשובות שגויות - בכל פעם שתכניסו סיסמה שגויה ירדו לכם 100 נקודות.3) רמזים - בכל פעם שתבקשו רמז ירדו לכם 100-500 נקודות (תלוי ברמז).<span class = "headline">תחתית המסך</span>בצד ימין - מופיע הזמן שעבר מתחילת החדר. הזמן יתחיל ברגע שבו תכנסו לחדר. הזמן יסתיים כשתסיימו את החדר.בצד שמאל - מופיע כפתור שמאפשר להתחיל ולהפסיק את מנגינת הרקע.במרכז - הודעות והתראות.
